# سوق لوميير - تطبيق عرض المنتجات والأسعار

تطبيق واجهة أمامية عربي كامل لعرض المنتجات والصور والأسعار بتجربة ثلاثية الأبعاد تفاعلية.

## الصفحات

- `/` الصفحة الرئيسية
- `/products` كتالوج المنتجات مع بحث وفلترة وترتيب وترقيم صفحات
- `/products/:id` صفحة تفاصيل المنتج
- `/offers` العروض
- `/about` عن المتجر
- `/contact` التواصل وإرسال طلبات المنتجات
- `/cart` سلة المشتريات

## الملفات

- `index.html` هيكل التطبيق والصفحات
- `style.css` التصميم، الحركة، العمق البصري والاستجابة
- `app.js` بيانات المنتجات، التوجيه، السلة، الفلاتر والتفاعلات
